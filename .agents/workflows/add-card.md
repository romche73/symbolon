# Add Symbolon Card
## Description
Ищет указанную карту в Symbolon.txt и автоматически добавляет её в index.html, настраивая картинку и делая коммит.

## Steps
1. Спроси у пользователя название карты, которую он хочет добавить (например: "Фортуна", "Цепи" или "Страх").
2. Открой и внимательно прочитай файл Symbolon.txt. Найди в нем раздел, соответствующий этой карте.
3. Извлеки из текста карты название и первый абзац из секции "Карта на день" (Что для меня важнее всего сегодня сделать?).
4. Открой файл index.html и аккуратно допиши эту карту в массив JavaScript. Путь к картинке укажи в формате "images/английское_название_карты.jpg".
5. Убедись, что код index.html остался рабочим и структура JS-массива не нарушена.
6. Сделай коммит в Git с автоматическим комментарием "feat: add card {название_карты} via workflow".
