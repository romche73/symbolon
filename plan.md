# План реализации: Карта дня Симболон

## Шаг 1: Базовая структура (Скелет сайта)
- Создать один файл `index.html` в папке проекта.
- Написать в нем стандартный базовый код HTML-страницы.
- Разместить на странице три элемента: заголовок "Карта дня Симболон", пустой текстовый блок (куда ИИ потом выведет карту) и кнопку "Вытянуть карту".

## Шаг 2: База данных карт (JavaScript)
- Встроить в файл `index.html` (в специальный раздел для скриптов) базу данных из 5 карт Симболон, описанных в `spec.md`.
- У каждой карты в коде должно быть название и текст толкования.

## Шаг 3: Логика выбора карты
- Написать логику (на языке JavaScript), которая будет запускаться при нажатии на кнопку "Вытянуть карту".
- При нажатии код должен выбирать одну случайную карту из пяти и отображать её название и текст в текстовом блоке на экране.

## Шаг 4: Базовое оформление (CSS)
- Добавить на страницу стили, чтобы сайт выглядел аккуратно.
- Расположить все элементы ровно по центру экрана.
- Оформить карточку с предсказанием: сделать красивую рамку, добавить тени и приятный шрифт, чтобы на экране это выглядело эстетично.
