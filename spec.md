# Спецификация проекта: «Карта дня Симболон»

## 1. Описание приложения
Простое одностраничное веб-приложение (HTML + CSS + JavaScript в одном файле), которое позволяет пользователю получить случайную «Карту дня» из колоды Симболон.

## 2. Функционал (Что делает приложение)
- На экране отображается красивая кнопка «Получить карту дня».
- При нажатии на кнопку приложение случайным образом выбирает одну из 5 доступных карт.
- На экране появляется название карты и текстовое толкование (ответы на вопросы дня).

## 3. База данных (5 карт Симболон)
Мы используем готовые тексты из файла проекта:
1. **Актер**: "Сегодня вы должны сыграть какую-то роль (короля, начальника, ребенка и т.д.)" [5].
2. **Свадьба**: "Хорошее настроение. Приятная компания. Удачный день" [6].
3. **Вестник**: "В этот день вас ожидают перемещения, короткие поездки, много общения, контактов..." [10].
4. **Симболон**: "Вам сегодня будет сопутствовать удача, особенно в коллективных делах..." [11].
5. **Черная Месса**: "В сегодняшних мероприятиях вам отводится роль пешки" [12].

## 4. Дизайн и ограничения
- Всё пишется в одном-единственном файле `index.html` для максимальной простоты.
- Дизайн самый базовый, аккуратный и минималистичный: кнопка по центру, карточка с текстом под ней.
- Никаких сложных внешних библиотек и баз данных.
